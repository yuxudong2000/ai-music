## 跨性别转换问题（P-6）

| 项目 | 说明 |
|------|------|
| source 性别 | 女（王菲，F0 均值 ~280Hz） |
| target 性别 | 男（痛仰，F0 均值 ~140Hz） |
| 输出听感 | **仍偏女声** |
| 根因 | seed-vc SVC 保留 source 的 F0 轮廓，auto-f0-adjust 只做整体平移，F0 仍处于女声区间 |

**解决方案**：

| 方案 | 说明 | 推荐度 |
|------|------|-------|
| A：`--semi-tone-shift -8~-10` | 强制降调，快速验证 | ⭐⭐⭐ |
| B：换用同性别 source/target | 根本避免跨性别问题 | ⭐⭐⭐⭐ |
| C：fine-tune 模式 | 训练时对齐音域分布 | ⭐⭐⭐⭐⭐（长期） |

## 结论

1. ✅ **bleedless 二次净化方案可行**，对去除伴奏泄漏有帮助
2. ❌ **跨性别 Zero-shot 转换效果不佳**，需配合降调或改用同性别对
3. 后续实验建议先用同性别 source/target 对验证净化效果的提升，再单独验证跨性别降调方案
